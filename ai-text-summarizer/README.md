AI Text Summarizer

A simple AI learning project.

This is a project I built while learning about AI and local LLMs.
The program sends text to a local model using Ollama and streams the summarized result in real time.

Tech

    * Python
    * Ollama
    * gpt-oss:20b

Run

    ollama serve
    python main.py

This project is part of my journey learning AI and building simple AI tools.
